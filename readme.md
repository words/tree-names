Common names for trees.

## Install

[npm][]:

```sh
npm install tree-names
```

## Use

```js
var trees = require('tree-names')

console.log(trees.slice(0, 10))
```

```js
[
  'acacia',  'alder',
  'apple',   'arborvitae',
  'ash',     'aspen',
  'avocado', 'balsam',
  'bamboo',  'banana'
]
```

## API

### `treeNames`

`tree-names` returns a list of strings (`Array.<string>`).
